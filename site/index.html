All Preview Datasets include primary data (fastq and metadata). Additional analysis of the Preview Datasets will be made available here as donated by the data generators.

Download Instructions for Project Data

1. Install the HCA Command Line Interface (HCA CLI)

The HCA CLI is used to download the data files listed in the project's file manifest. Install the HCA CLI by following the HCA CLI installation instructions.

2. Obtain the preview project's file manifest

Follow any of the "Project Data File Manifest" links above to select the desired project in the HCA Data Browser. In the Data Browser, select the "Request File Manifest" button, then select the desired file types and generate and download the project's file manifest.

3. Use the HCA CLI to download the files listed in the project's file manifest

With the HCA CLI installed, and the project's file manifest downloaded, use the HCA CLI to download the files listed in the project's file manifest.
