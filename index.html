These include primary datasets (fastq and metadata). Additional analysis (alignment and quantification) produced from a standardized analysis pipeline will be made available this summer. If you have questions please email data-help@humancellatlas.org.

Download instructions

1. Download the shell script above

Click on the download button to save a shell script that will download HCA data. The shell script requires a Unix environment (Linux, OS X, Windows WSL, or Windows Cygwin), as well as a Python installation with a functional pip installer.

2. Open it in the terminal and run

After saving the script, run it in your terminal as follows:

chmod +x ~/Downloads/FILENAME.sh

Next, run:

~/Downloads/FILENAME.sh

The shell script will install the HCA command line client using the pip installer, then call the client to download a list of HCA DSS bundle IDs.

3. Open the script in a text editor for instructions

Open the script in a text editor to see the list of IDs and download commands.
